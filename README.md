# Senior Frontend Developer Test Assignment

## Overview

This is a simple subscription application with login and payment flow. Your task is to identify and fix bugs in the application.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Copy environment file:
```bash
cp .env.example .env
```

3. Start the development server:
```bash
npm run dev
```
